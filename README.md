Programming Language : python 3.6 이상  
Language : Korean

## Get Started
1. Install requirement
```
pip install -r requirement.txt
```

2. Create `.env` File
```
HERREN_TOKEN=YOUR_TOKEN
```

3. Run `example_component.py`
